MESSAGE SERVER
For use of the messageServer.js file, follow these steps to run the server:
1. Download Node.js
2. Run Node.js command prompt as admin
3. Run the command 'npm install -g express'
4.Navigate to the folder where the "messageServer.js" file is sored by running 'cd' commands
5. Install the WebSocket with this command: 'npm install websocket'
 NOTE:WEBSOCKET SHOULD BE INSTALLED IN THE SAME FOLDER CONTAINING THE SERVER.
6. Run this command to now connect to the server 'node messageServer.js'
7. The Server will alert you whenever a new Device is Connected.
